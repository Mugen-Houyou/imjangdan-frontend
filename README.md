# 땅판소년단's 임장단 프론트엔드

## 권장 사항
- Windows 10 이상 또는 macOS 14.0 이상

- Node.js 18 이상

## 설치 및 실행

`npm i`

`npm run start`



## 기반 프로젝트: PaathShaala
https://codesandbox.io/s/react-4wom5
